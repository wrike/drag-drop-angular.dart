Drag Drop Example 12: Behavior for images, links and input element inside and outside managed draggable container

Images and links always goes as dragable element by default in all major browsers, and input element do not prevent drag event.

Inside managed draggable container default behavior of these elements is prevented, except navigation to links. Input elements are properly focused.
